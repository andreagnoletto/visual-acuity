# Fonte OpticianSans

A fonte OpticianSans.woff2 precisa ser baixada e colocada neste diretório.

## Como baixar:

1. Acesse: https://optician-sans.com
2. Clique em "Download for free"
3. Extraia o arquivo ZIP
4. Copie o arquivo `OpticianSans-Regular.woff2` para este diretório como `OpticianSans.woff2`

Ou baixe diretamente do GitHub:
- Repositório: https://github.com/anewtypeofinterference/Optician-Sans
- Caminho: `fonts/woff2/OpticianSans-Regular.woff2`
- Renomeie para: `OpticianSans.woff2`
